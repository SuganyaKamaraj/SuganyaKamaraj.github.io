Projects
Embedded System

KCET

Ultrosonic Spectacle and Waist-Belt for Visually Imapired and Blind Person
This project is an electronic navigation system for visually impaired and blind people (subject). This system understands obstacles around the subject up to 500 cm in front, left and right direction using a network of ultrasonic sensors. It effectively calculates distance of the detected object from the subject and prepares navigation path accordingly avoiding obstacles. It uses speech feedback to aware the subject about the detected obstacle and its distance. This proposed system uses AT89S52 microcontroller based embedded system to process real time data collected using ultrasonic sensor network. Based on direction and distance of detected obstacle, relevant pre-recorded speech message stored in APR9600 flash memory is invoked. Such speech messages are conveyed to the subject using earphone.

Machine Learning

MEPCO

Diabetic Eye Diseases Detection and Classification using Support Vector Machine and Random Forest Classifier
Diabetic eye disease is one of the major problem. Early detection of eye diseases can increase the chance of successful treatment and also survival rate. The proposed method is to explore machine learning techniques to detect diabetic eye diseases using thermography images of an eye and to introduce the effect of thermal variation of abnormality in the eye structure as a diagnosis imaging modality which are clinically helpful for both patients and ophthalmologists. The thermal images are pre-processed, features are extracted and classified using classifier with various combination of features. To detect abonormality, two supervised machine learning technique like Support Vector Machine and Random Forest are described and their performance are compared. Experimental results for RF has a maximum accuracy of 89.37% and SVM has a maximum accuracy of 85.49%.

Diabetic Eye Diseases Detection and Classification using Deep Convolutional Neural Network
Diabetic Eye Disease (DED) is a common complication of diabetes and one of the major causes of blindness in the active population. The early detection of DED occurrence can be very helpful for clinical treatment. Recently, deep convolutional neural networks have manifested superior performance in image classi?cation compared to previous handcrafted feature-based image classi?cation methods. In this project, I proposed a deep Convolutional Neural Network (CNN) methodology for the automatic classi?cation of diabetic eye diseases using thermal image, and obtained an accuracy of 95.38% on our dataset, outperforming the results obtained by using classical approaches. The obtained preliminary results clearly demonstrate that the proposed deep learning method is promising in automatic diagnosis of diabetic eye diseases.
